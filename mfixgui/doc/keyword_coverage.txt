   Set keyword BC_HW_X_S(#,#,#) to 0.0
   Set keyword BC_C_X_S(#,#,#) to 0.0

However, the GUI does not provide any way for these keys to be set to any nonzero value, or UNDEFINED

A few tutorials set this key, but only to 0, see
 SpoutedBedCombustor and silane_pyrolysis
